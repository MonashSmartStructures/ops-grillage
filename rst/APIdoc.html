GrillageMember class¶

class members.GrillageMember(section: members.Section, material, member_name='Undefined', quad_ele_flag=False, tri_ele_flag=False)¶

Bases: object

Main class for Grillage member definition. Class requires two objects as input parameters: a Section object and a Material object.

Parameters

  • section (Section) – Section class object assigned to GrillageMember

  • material (uniaxialMaterial) – Material class object assigned to GrillageMember

  • name – Name of the grillage member (Optional)

This module manages user interface functions and class definitions for the grillage members. In our terminology, we define a section as the geometrical properties of the structural elements, and the member as the combination of the section and material properties.
