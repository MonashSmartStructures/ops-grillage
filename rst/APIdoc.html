class members.Section(op_ele_type='elasticBeamColumn', mass=0, c_mass_flag=False, unit_width=False, op_section_type='Elastic', **kwargs)¶

Bases: object

Class to wrap Openseespy command to create sections.

Parameters

  • op_ele_type (str) – Opensees element type

  • op_section_type (str) – Opensees section type

  • unit_width (bool) – Flag for unit width properties

Here are some basic keywords for Section. Section accepts Openseespy keyword arguments for various element properties as well - see Openseespy section() and element()

Keyword

  • A (float): Cross sectional area

  • Iz (float): Moment of inertia about local z axis

  • Iy (float): Moment of inertia about local y axis

  • J (float): Torsional inertia - about local x axis

  • Az (float): Cross sectional area in the local z direction

  • Ay (float): Cross sectional area in the local y direction

GrillageMember class¶

This module manages user interface functions and class definitions for the grillage members. In our terminology, we define a section as the geometrical properties of the structural elements, and the member as the combination of the section and material properties.
