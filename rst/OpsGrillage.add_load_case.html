OpsGrillage.add_load_case¶

OpsGrillage.OpsGrillage.add_load_case(self, load_case_obj: Union[Load.LoadCase, Load.CompoundLoad], load_factor=1)¶

Function to add individual load cases to OpsGrillage instance.

Parameters

  • load_factor – Optional load factor for the prescribed load case

  • load_case_obj (LoadCase) – A load case object of the load condition

A typical load_case_dict within load_case_dict list has the following format .. code:

load_case_dict = {‘name’:load_case_obj.name, ‘loadcase’: load_case_obj, ‘load_command’: load_str,

‘load_factor’: load_factor}
